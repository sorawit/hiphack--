HipWatcher
---------

HipServer
---------
0. Run HipWatcher! Now!
1. Make sure you have vagrant system setup (https://www.virtualbox.org/wiki/Downloads, https://www.vagrantup.com/downloads.html)
2. cd hiphack
3. vagrant up && vagrant provision
4. visit localhost:9000
